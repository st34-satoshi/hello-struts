# hello-struts

## Prepare
- Install Java: https://www.oracle.com/java/technologies/downloads/
- Install Maven: https://maven.apache.org/download.cgi

## Run
- `mvn jetty:run`
- open: `http://localhost:8080/hello-struts/index.jsp`
